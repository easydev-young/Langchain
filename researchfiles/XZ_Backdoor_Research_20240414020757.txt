XZ backdoor is a maliciously introduced backdoor in the Linux utility xz within the liblzma library. It gives an attacker remote code execution capabilities on the affected Linux system. The issue has been assigned a CVE number CVE-2024-3094 and has a CVSS score of 10.0. XZ Utils is a set of free software command-line lossless data compressors that includes programs like lzma and xz. A backdoor is a covert method of bypassing normal authentication or encryption in a computer system.
